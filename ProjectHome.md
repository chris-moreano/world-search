The goal of the program is:
Reading the input file and putting the data in the appropriate data structures.
Considering the words one by one
Searching in the table horizontally, vertically, diagonally in 8 different directions
Saving the address of the location of each word as two ordered-pairs (row, col), one for the start and one for the end.
Keep searching to find all locations that the word is located.
Reporting the word and its locations as the example will show you.
The program is going to be designed neatly and will be implemented using the golden rules of programming (that much that you have seen so far.)
